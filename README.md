# جيو الرافدين — النسخة الأولية

واجهة عربية متجاوبة لمنصة الخدمات الجغرافية والاستشعار عن بُعد، تشمل:

- عرض الخدمات ومصادر البيانات.
- نموذج طلب خدمة مع تحقق من الحقول.
- تسجيل دخول آمن عبر رابط بريد إلكتروني دون كلمة مرور.
- قاعدة بيانات محمية بسياسات وصول على مستوى الصفوف.
- تخزين خاص للملفات ولوحة للعميل والمدير.
- خريطة تفاعلية لمحافظات العراق لتحديد منطقة الدراسة.
- متابعة إدارية للسعر ونسبة الإنجاز والتسليم ورسالة العميل.
- رفع ملفات GeoJSON وGeoPackage وملفات الشيب فايل المضغوطة بصيغة ZIP ضمن حدود آمنة.
- سياسة خصوصية واحتفاظ لمدة 90 يوماً، وسجل تدقيق لإجراءات المدير.
- مسار مصادقة ثنائية للمدير وتجهيز Cloudflare Turnstile.

## التشغيل

بعد إكمال [`SETUP.md`](SETUP.md)، انقر مرتين على:

`Start-GeoRafidain.cmd`

ستفتح المنصة على `http://127.0.0.1:8765`. أبقِ نافذة التشغيل مفتوحة أثناء الاستخدام.

يمكن فتح `index.html` لمعاينة التصميم فقط، لكن تسجيل الدخول والطلبات محجوبان عبر `file://` لأسباب أمنية.

## حالة النسخة الحالية

المشروع مربوط بـ Supabase، وترحيلات الحماية وملفات GIS مطبقة ومتحقق منها، ويستخدم نسخة محلية ثابتة من `supabase-js 2.108.2`. لا تضف أي مفتاح Secret أو Service Role إلى الواجهة.

قبل النشر العام يلزم نطاق HTTPS، وتفعيل Turnstile/SMTP، ونشر مهمة الحذف الموجودة في `RETENTION.md`، ثم اختبار بحساب عميل منفصل.
